
# Tour Weaver v1.3.3

Plataforma profesional para la creación y exhibición de tours virtuales 360°.

## Características principales
- Creación de tours interactivos con escenas 360°.
- Registro de unicidad de URLs (Slug Registry).
- Panel de analíticas avanzado.
- Compartir mediante Código QR descargable.
- Verificación dinámica de Google Search Console y Analytics.

Desarrollado con Firebase Studio.
